# Программа для обращения (реверсирования) масссива
## Описание
Эта программа на С++ переворачивает элементы массива в обратном порядке. Исходный массив изменяется "на месте"(in-place) без создания дополнительного массива.
## Функциональность 
Принимает исходный массив целых чисел
Реверсирует порядок элементов в массиве
Выводит результат на экран
## Пример работы 
Выходные данные:{а,б,в,г,д}
Входные данные:{д,г,в,б,а}
